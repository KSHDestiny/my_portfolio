## Best Practice For The CV

- Use only 1 link per project entry in the resume itself.
- Link the project name to the dedicated `/projects/<slug>` page.
- Keep diagram and UI links for LinkedIn, cover letters, or interviews.
- If space is tight, link only your main portfolio plus 2 to 4 strongest project pages.
