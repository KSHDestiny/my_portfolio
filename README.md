## Tech Stack
- Framework: Next.js 15 (App Router), React 19
- Language: TypeScript
- Styling: Tailwind CSS + shadcn/ui
- Animation: Framer Motion + tsparticles
- Form delivery: EmailJS
- PDF preview: embedded browser viewer (`iframe`)

## Key Portfolio Features
- Responsive single-page section layout
- Dark/light theme toggle
- Animated project cards and section transitions
- Certificate modal with in-app preview + open/download actions
- Updated CV download from `public/Kaung-Sat-Hein-CV.pdf`

## Local Development
```bash
npm install
npm run dev
```

Open: `http://localhost:3000`

## Build
```bash
npm run build
npm run start
```

## Project Structure
```text
app/                 # App Router entry (layout + page)
components/          # Portfolio sections and shared components
components/ui/       # shadcn/radix UI primitives
hooks/               # Custom React hooks
lib/                 # Utilities
public/              # Static assets, certificates, CV PDF
```
