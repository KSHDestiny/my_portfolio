## 🎯 Objective

Quick reference for:

- Image tagging

- Image creation & deletion

- Network creation & deletion

- Running containers with volume & port mapping

---

# 🏷️ Docker Tag (Image Re-tagging)

## 🧠 Concept

Create a new tag for an existing image (no duplication)

## ⚙️ Command

```plain text
docker tag <source_image>:<tag> <target_image>:<tag>
```

## 📌 Example

```plain text
docker tag app:latest app:custom
```

---

# 📦 Docker Image Creation

## 🧠 Concept

Create image from a container snapshot

## ⚙️ Command

```plain text
docker commit <container_name> <image_name>:<tag>
```

---

# 🗑️ Docker Image Deletion

## ⚙️ Command

```plain text
docker rmi <image_name>:<tag>
```

## ⚠️ If in use

```plain text
docker rm -f <container_name>
docker rmi <image_name>:<tag>
```

---

# 🌐 Docker Network Creation

## 🧠 Concept

Custom network for container communication

## ⚙️ Command

```plain text
docker network create \
--driver bridge \
--subnet <subnet> \
--gateway <gateway> \
  <network_name>
```

---

# 🗑️ Docker Network Deletion

## ⚙️ Command

```plain text
docker network rm <network_name>
```

## ⚠️ If in use

```plain text
docker network disconnect <network_name> <container_name>
docker network rm <network_name>
```

---

# 🚀 Run Container (Volume + Port Mapping)

## 🧠 Concept

- v → host ↔ container file sync

- p → expose container service

---

## ⚙️ Command

```plain text
docker run-d \
--name <container_name> \
-p <host_port>:<container_port> \
-v <host_path>:<container_path> \
  <image_name>:<tag>
```

---

## 🔍 Verify

```plain text
docker ps
docker inspect <container_name> | grep -A 10 Mounts
docker port <container_name>
curl http://localhost:<host_port>/
```
