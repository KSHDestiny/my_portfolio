### Key Idea

Launch an Amazon EC2 (Elastic Compute Cloud) instance to create a virtual server in the AWS cloud.

### What Matters

- Web servers
- Application servers
- DevOps environments
- Development and testing systems
- Sign in to AWS

### Quick Summary

EC2 instances allow users to run applications, host websites, and perform computing tasks on scalable infrastructure. Amazon EC2 provides resizable compute capacity in the cloud.
