### Key Idea

Assign a static public IP (Elastic IP) to an EC2 instance so it remains reachable even after stop/start.

### What Matters

- ✅ Persistent (does not change)
- ✅ Can be reattached to another instance
- ❌ Charged if not in use
- Go to EC2 Dashboard
- Click Elastic IPs (left sidebar)

### Quick Summary

An Elastic IP (EIP) is a static IPv4 address provided by AWS. 👉 EIP is associated directly with the instance or its network interface (ENI).
