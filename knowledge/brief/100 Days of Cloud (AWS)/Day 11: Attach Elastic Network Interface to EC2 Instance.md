### Key Idea

Attach an Elastic Network Interface (ENI) to an EC2 instance to enable:

### What Matters

- Multiple IP addresses
- Network separation
- Advanced networking setups
- Private IP address (primary + secondary)
- Optional Elastic IP

### Quick Summary

An Elastic Network Interface (ENI) is a virtual network card in AWS. 👉 An EC2 instance can have multiple ENIs attached.
