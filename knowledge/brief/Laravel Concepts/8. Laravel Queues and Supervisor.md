### Key Idea

I use Laravel Queues with Supervisor to handle background jobs reliably and keep APIs fast.

### What Matters

- application receives request
- dispatches a job using Laravel
- queue stores the job (Redis / DB / SQS)
- worker picks up the job
- job is processed asynchronously

### Quick Summary

Instead of doing heavy work inside a request: 👉 I push the task to a queue
