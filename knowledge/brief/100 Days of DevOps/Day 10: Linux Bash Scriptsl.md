### Key Idea

Learn how to create, write, and execute Bash scripts in Linux to automate tasks such as running commands, managing files, and scheduling operations.

### What Matters

- Automating system administration tasks
- Running backups
- Deploying applications
- Monitoring systems
- Running scheduled jobs (cron)

### Quick Summary

A Bash script is a file containing a series of commands that the Bash shell executes sequentially. Instead of typing commands one by one in the terminal, you can automate them inside a script.
