### What a Bash Script Is

A Bash script is a file that contains shell commands executed in sequence by the Bash interpreter. It is used to automate repeated command-line tasks and make workflows easier to reuse.

### Run a Bash Script

Use this command to run a script manually:

```bash
bash script.sh
```

### Bash Shebang

Use this line at the top of the script file:

```bash
#!/bin/bash
```

### Simple Script Command

A basic command inside a Bash script looks like this:

```bash
echo "Hello, World"
```

## Summary

Bash scripts help automate Linux tasks and reduce repeated manual command entry.
