### Key Idea

Create a Virtual Network (VNet) in Azure to enable secure communication between Azure resources such as Virtual Machines, databases, and services.

### What Matters

- Communicate securely with each other
- Connect to the internet
- Connect to on-premises networks (VPN or ExpressRoute)
- Sign in to Azure
- Go to the Azure Portal: https://portal.azure.com

### Quick Summary

A VNet allows resources to communicate privately within Azure, and optionally with on-premises networks or the internet. A Virtual Network (VNet) is a logically isolated network in Azure, similar to a traditional network in a data center.
