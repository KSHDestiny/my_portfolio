### Key Idea

Attach an additional Network Interface Card (NIC) to an Azure Virtual Machine to enable advanced networking and traffic separation.

### What Matters

- Private IP address
- Public IP (optional)
- Network Security Group (NSG)
- Subnet association
- 🔹 Separate public & private traffic
