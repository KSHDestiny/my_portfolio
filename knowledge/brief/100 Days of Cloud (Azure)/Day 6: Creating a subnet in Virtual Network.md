### Key Idea

Create a subnet inside an Azure Virtual Network (VNet) to divide the network into smaller segments for better organization, security, and resource management.

### What Matters

- VNet: 10.0.0.0/16
- Subnet: 10.0.1.0/24
- Sign in to Azure
- Go to the Azure Portal: https://portal.azure.com
- Navigate to Virtual Networks

### Quick Summary

Subnets let Azure resources like Virtual Machines, databases, and services live in separate network segments. A subnet is a smaller network created inside a Virtual Network (VNet).
