### Key Idea

Create and assign a Public IP address to an Azure Virtual Machine so it can be accessed from the internet.

### What Matters

- Remote access (SSH / RDP)
- Hosting web applications
- External communication with cloud resources
- Subscription → Select your subscription
- Resource Group → Select existing resource group

### Quick Summary

Public IPs allow users to connect to the VM using services like SSH, RDP, or web applications. A Public IP address allows Azure resources such as Virtual Machines to communicate with the internet.
