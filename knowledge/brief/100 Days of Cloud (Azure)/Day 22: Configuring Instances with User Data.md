### Key Idea

Automatically configure an instance at launch using User Data scripts.

### What Matters

- User Data → Script that runs when an instance starts (first boot)
- Used for:
- Installing packages
- Starting services
- Configuring environment

### Quick Summary

👉 Saves time → no manual setup after launch User Data allows you to automate instance setup during launch, enabling consistent, repeatable, and fast provisioning without manual intervention.
